## Design Principles

- Keep interfaces clean and uncluttered. Use generous white space.
- Prefer warm cream backgrounds over stark white.
- Maintain consistent spacing and alignment.
- Use green as the dominant brand color.
- Use mustard only for highlights.
- Use chili red sparingly for calls-to-action and promotional elements.
- Avoid gradients in the logo; use them only as subtle website accents.
- Favor rounded corners, soft shadows, and high-contrast typography.
- Ensure all UI components meet accessibility guidelines with sufficient color contrast.
